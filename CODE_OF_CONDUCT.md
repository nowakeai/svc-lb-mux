# Code of Conduct

Service LoadBalancer Multiplexer is an open-source infrastructure project under the nowake.ai GitHub organization. We want discussions to stay technical, respectful, and useful for people operating real systems.

## Expected Behavior

- Be direct, factual, and respectful.
- Assume good intent while still asking for evidence when claims affect correctness or safety.
- Keep issues and pull requests focused on the project.
- Document operational context when reporting bugs or provider behavior.
- Respect maintainers' time by providing reproduction steps, versions, logs, and relevant manifests.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.
- Filing security issues publicly before maintainers have had a chance to investigate.
- Abusive automation, spam, or low-signal generated content.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or restrict participation when behavior harms the project or community.

For security issues, follow [SECURITY.md](SECURITY.md). For support expectations, follow [SUPPORT.md](SUPPORT.md).
