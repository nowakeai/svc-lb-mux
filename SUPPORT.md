# Support

Service LoadBalancer Multiplexer is an open-source project. Public support happens through GitHub issues and discussions when enabled.

## Best-Effort Support

Use GitHub issues for:

- reproducible bugs;
- documentation problems;
- provider-specific validation reports;
- feature proposals;
- Helm chart or install issues.

Include:

- controller image tag or commit;
- Kubernetes version and provider;
- Helm values with secrets removed;
- relevant Service, Endpoints, ConfigMap, and event snippets;
- expected behavior and actual behavior;
- whether GitOps is managing the mux Service.

## Not Covered

The public issue tracker is not a production incident response channel. Do not post secrets, private cluster credentials, internal hostnames, or sensitive customer data.

## Security

Report vulnerabilities through the process in [SECURITY.md](SECURITY.md), not public issues.
