# Security Policy

Please do not report suspected vulnerabilities in public issues before maintainers have had a chance to investigate.

Use GitHub private vulnerability reporting if it is enabled for this repository. If it is not available, contact the maintainers privately and include:

- affected version or commit
- Kubernetes provider and version, if relevant
- reproduction steps
- impact assessment
- any known mitigations

We will acknowledge valid reports and coordinate fixes before public disclosure.
